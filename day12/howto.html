Gemini VS DeepSeek 🤖💡

이 프로젝트는 Gemini와 DeepSeek을 활용하여
AI 기반의 질문-답변 시스템을 구축하는 웹 애플리케이션입니다. 🌐✨

🚀 기술 스택

HTML5 CSS3 JavaScript LocalStorage Fetch API Proxy

📌 주요 기능

  • 🤖 AI 모델 선택: Gemini 또는 DeepSeek을 선택하여 답변 생성 가능
  • 📩 질문 등록: 사용자가 입력한 질문을 저장하고 AI가 답변 제공
  • 🔄 데이터 저장: LocalStorage를 활용한 영속성 관리
  • 🗑️ 삭제 기능: 개별 질문 삭제 및 전체 초기화
  • ⚡ 비동기 요청: Fetch API를 활용하여 AI API 호출

🛠️ 사용 방법

  1. 웹 페이지를 실행합니다. 🌍
  2. 질문을 입력하고 원하는 AI 모델을 선택합니다. 🎯
  3. 등록 버튼을 눌러 질문을 제출합니다. 🚀
  4. AI의 답변을 확인하고, 필요하면 삭제할 수 있습니다. ✨
